Invoer en uitvoer

Invoer (Input) is de code dat staat voor een plek waar je text in kan voeren. Dit wordt bijvoorbeeld veel gebruikt
bij dingen zoals vragenlijsten, die vul je in. In deze code staat dan een input waar je in kan typen
